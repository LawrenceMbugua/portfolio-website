Task-tracking system

This was a project I started back in January 2022 when I first learned React. I was the sole creator/developer in the initial stages of the project before I was able to hire help.

The system enables users to keep track of their daily tasks thus better time management.

Tasks are saved along with their duration in the database.

Users have the ability to edit their tasks whenever they feel like doing so.

Technologies:
  • - Javascript
  • - Node.js + Express Framework
  • - React JS
  • - MongoDB
  • - Bootstrap
